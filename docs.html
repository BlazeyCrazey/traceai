TraceAI Documentation

Everything you need to design, validate, and manufacture PCBs with TraceAI.

Overview

TraceAI is a browser-based PCB design tool that combines a professional schematic and layout editor with an AI assistant powered by Claude. You can go from idea to production-ready Gerber files without installing any software.

The editor runs entirely in your browser. Projects are saved to the cloud automatically, so you can pick up your work from any machine.

Tip

TraceAI works best in Chrome, Edge, or Firefox. Safari is supported but WebGL performance may vary on older macOS versions.

Create an Account

Head to the sign-up page and create an account with your email or sign in with Google or GitHub. Free accounts include unlimited public projects and up to 3 private projects.

  1. Click Get Started in the top-right corner.
  2. Enter your email and choose a password, or click the Google / GitHub button.
  3. Verify your email address (check your spam folder if you don't see it).
  4. You'll be redirected to your project dashboard — you're ready to go.

Your First Project

From the dashboard, click New Project. Give it a name and optional description, then click Create Project. The editor opens with an empty schematic sheet and the component browser on the left.

A quick way to get started is to type a request into the AI chat panel on the right:

Create a simple LED blinker circuit using a 555 timer

The AI will place components, wire them together, and explain the circuit. You can then modify the design or move directly to PCB layout.

Editor Overview

The editor is split into four areas:

  • Toolbar (top) — Mode selection, undo/redo, zoom controls, and design rule check.
  • Component Browser (left) — Searchable library of schematic symbols and PCB footprints.
  • Canvas (center) — The main drawing area for schematics and PCB layouts.
  • AI Panel (right) — Chat interface for the AI assistant. Collapsible.
Tip

Press Tab to cycle focus between the canvas and the AI chat input, so you can alternate between drawing and asking questions without touching the mouse.

Canvas Navigation

The canvas supports standard pan and zoom gestures:

  • Pan — Middle-click drag, or hold Space and left-click drag.
  • Zoom — Scroll wheel, or pinch on a trackpad. The canvas zooms toward the cursor.
  • Fit to view — Press F to frame all objects, or Shift+F to frame the selection.

Toolbar

The toolbar changes depending on whether you are in Schematic or PCB mode. Common actions are always available:

  • Select (V) — Default mode. Click to select, drag to box-select.
  • Wire / Trace (W) — Draw wires (schematic) or traces (PCB).
  • Place Component (P) — Opens the component search palette.
  • Measure (M) — Measure distance between two points on the canvas.
  • Undo / Redo — Ctrl+Z / Ctrl+Shift+Z.

Placing Components

Press P or click the Place Component button to open the search palette. Type a part name, value, or package description — for example "100nF 0402" or "ATmega328P".

  1. Select a component from the results list. A preview attaches to your cursor.
  2. Press R to rotate before placing.
  3. Click on the canvas to place the component. Click again to place another copy, or press Esc to finish.

Editing Properties

Double-click any component to open its property editor. You can change:

  • Reference Designator — e.g., R1, C5, U3.
  • Value — e.g., 10kΩ, 100nF, LM7805.
  • Footprint — Switch between footprint variants (e.g., 0402, 0603, 0805).
  • Custom fields — Manufacturer part number, supplier, notes.

Custom Components

If you need a part that isn't in the built-in library, you can create one:

  1. Open Library Manager from the sidebar.
  2. Click New Component and choose Symbol + Footprint.
  3. Draw your schematic symbol using the built-in shape tools (lines, arcs, pins).
  4. Use the footprint wizard to define pads from a datasheet drawing, or import a KiCad .kicad_mod file.
  5. Save the component to your personal library — it will appear in the search palette from now on.
Tip

You can also ask the AI: "Create a custom component for [part number]" and it will generate the symbol and footprint from the datasheet automatically.

Creating Schematics

Every project starts with a schematic. The schematic captures the logical connections between components before you think about physical layout.

  1. Place your main IC or module first — it anchors the design.
  2. Add passive components (resistors, capacitors, decoupling caps).
  3. Wire everything together, using net labels for power rails and long-distance connections.
  4. Run Electrical Rules Check (ERC) to catch missing connections and conflicting pin types.

Wiring

Press W to enter wire mode. Click a component pin to start a wire, click intermediate points to create bends, and click a destination pin to complete the connection. Wires snap to the grid and automatically form junctions when they cross existing wires at the same net.

To cancel a wire in progress, press Esc. To delete a wire segment, select it and press Delete.

Net Labels

Net labels connect pins that share the same name without drawing a visible wire. This keeps complex schematics readable.

  • Press L to add a net label. Type a name (e.g., VCC, SDA, MOSI) and place it on a wire or pin.
  • Use power symbols (VCC, GND, 3V3) from the component palette for supply rails.
  • All pins and wires sharing the same net label are electrically connected.

Board Setup

Switch to PCB mode from the editor tab bar. If you haven't set up the board yet, the wizard will guide you through:

  1. Board outline — Draw a rectangular or custom-shaped outline, or enter exact dimensions.
  2. Layer count — Choose 2, 4, or more layers. Default is 2-layer.
  3. Design rules — Set minimum trace width, clearance, via size, and other constraints based on your manufacturer.

Components from the schematic appear as an unplaced "ratsnest" next to the board. Drag them onto the board and arrange them logically before routing.

Routing

Press W in PCB mode to start routing traces. Click a pad to begin, route through the board, and click the destination pad to finish.

  • The interactive router pushes and shoves existing traces to make room.
  • Press V during routing to insert a via and switch layers.
  • Hold Shift to route in 45-degree increments.
  • For differential pairs, select both nets and use Route Differential Pair from the toolbar.

Layers & Stackup

Open Board Setup > Stackup to configure your layer structure. For a standard 4-layer board:

  • Top copper — Signal layer (components and traces).
  • Inner 1 — Ground plane.
  • Inner 2 — Power plane.
  • Bottom copper — Signal layer.

You can customize dielectric thickness and copper weight. The editor uses this information for impedance calculations and DRC.

Design Rule Check

Press Ctrl+Shift+D or click the DRC button to run a full check. The DRC verifies:

  • Clearance — Copper-to-copper, copper-to-edge, and drill-to-drill spacing.
  • Connectivity — All schematic nets are fully routed (no unconnected pads).
  • Annular ring — Via and pad annular rings meet minimum requirements.
  • Trace width — No trace is narrower than the minimum specified in design rules.
  • Silk overlap — Silkscreen does not overlap exposed copper.

Violations appear as markers on the canvas and in a list. Click a violation to zoom to it.

Using the AI Chat

The AI panel on the right side of the editor accepts natural-language requests. You can ask it to:

  • Create circuits from a description ("design a 5V to 3.3V regulator").
  • Explain how a selected sub-circuit works.
  • Suggest component values for a given specification.
  • Review your design for common mistakes.
  • Auto-route remaining unconnected nets.

The AI sees your current schematic and PCB state, so it can give context-aware answers.

Tips & Best Practices

  • Be specific: "Add a 100nF decoupling cap on U3 VCC pin" works better than "add a capacitor."
  • Ask the AI to explain before you override its suggestions — it may have considered constraints you missed.
  • Use /review after major changes. The AI catches issues like missing pull-ups, incorrect bypass cap placement, and thermal relief problems.
  • The AI respects your design rules. If it can't route a trace within your constraints, it will tell you what to change.
Tip

You can reference components by their designator in chat: "Move R12 closer to U3 pin 7" or "What is C4 for?"

File Formats

TraceAI can export your design in several formats:

  • Gerber RS-274X — Industry-standard PCB fabrication files (one per layer).
  • Excellon drill — NC drill file for hole positions and sizes.
  • PDF — Printable schematic and layout sheets.
  • SVG — Scalable vector graphics for documentation.
  • BOM (CSV / Excel) — Bill of Materials with part numbers, quantities, and references.
  • Pick-and-place — Component centroid file for automated assembly.

Go to File > Export or use the /export command. The Gerber preview shows each layer so you can verify before downloading.

Manufacturer Integration

TraceAI integrates directly with popular PCB manufacturers:

  • JLCPCB — Get instant quotes, upload Gerbers, and order with assembly.
  • PCBWay — Direct order submission with automatic file validation.
  • OSH Park — Shared-run ordering for prototype quantities.

Click Order PCB in the export panel, choose your manufacturer, and configure options (layer count, finish, thickness). TraceAI generates the correct file package and opens the manufacturer's order page with your files pre-uploaded.

Tip

Set your preferred manufacturer in Project Settings and TraceAI will tailor DRC rules to match their capabilities, reducing the chance of fabrication issues.
